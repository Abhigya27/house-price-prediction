# House Price Prediction

A Flask web application that predicts California housing prices using a machine learning model trained on the California Housing dataset.

## Technologies Used
- Python
- Flask
- Scikit-Learn
- Pandas
- NumPy

## Features
- User-friendly web interface
- Real-time house price prediction
- Trained Linear Regression model

## Run Locally

```bash
pip install -r requirements.txt
python application.py
